Welcome to the spaceship game!
As you start the game, press enter to launch the rocket.
The rocket can be moved left, right, up or down.
The goal of the game is to reach the planet, in the bottom right corner of the game screen.
The player has three lives to achieve this goal.
There are two main rocks (red bricks), that if collided with the rocket, will result is loss of a life.
Every time the player collides with a rock, the rocket is sent back to the starting position.
If the player loses 3 lives, a lose screen appear. 
If the player reaches the planet before losing their lives, a win screen appears.
At any point in the game, the user can restart by clicking select / return key.
